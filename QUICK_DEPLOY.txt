🚀 QUICK DEPLOY CHECKLIST - Sales Analytics Dashboard
========================================================

✅ FILES READY FOR DEPLOYMENT:
- app.py (updated for production)
- requirements.txt (Flask, pandas, gunicorn)
- Procfile (tells Render how to run)
- runtime.txt (Python version)
- .gitignore (excludes unnecessary files)

📋 DEPLOY IN 5 STEPS:

1️⃣ CREATE GITHUB REPO
   → Go to: https://github.com/new
   → Name: sales-analytics-dashboard
   → Make it Public
   → Create repository

2️⃣ UPLOAD FILES TO GITHUB
   → Click "uploading an existing file"
   → Drag ALL project files and folders
   → Commit changes

3️⃣ SIGN UP ON RENDER
   → Go to: https://render.com
   → Sign up with GitHub

4️⃣ CREATE WEB SERVICE
   → Click "New +" → "Web Service"
   → Connect your GitHub repo
   → Build Command: pip install -r requirements.txt
   → Start Command: gunicorn app:app
   → Click "Create Web Service"

5️⃣ WAIT 3 MINUTES
   → Render builds and deploys automatically
   → You'll get a live URL!

🎉 DONE! Your dashboard is live at:
   https://sales-analytics-dashboard-XXXX.onrender.com
